# ATM Simulation System

## Description
This is java based ATM-Simulation-System developed using OOP concepts

## Features
-PIN Login
-Balance Check
-Deposit Money
-Mini statement
-Withdraw money

## Technologies used
-java
-OOP concept
-Arraylist
-Scanner class

## how to Run
1.compile the code :
" javac ATMSimulation.java ".
2.Run the code:
" java ATMSimulation "
